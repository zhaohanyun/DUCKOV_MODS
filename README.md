# DUCKOV_MODS

《逃离鸭科夫》（Escape from Duckov）游戏个人开发 Mod 集合

## Mod 列表

### InfiniteSyringes - 无限针剂

**版本**: 1.0.0  
**作者**: 不明嚼栗  
**描述**: 无限针剂 - 使用针剂类物品不消耗数量

**功能**:
- 打针类药品使用后自动恢复，永不消耗
- 支持多语言识别（繁体中文、简体中文、英文）
- 自动识别所有针剂类物品（止血针、硬化针、负重针、镇痛剂等）
- 精确区分"使用"和"拆分"操作，不影响正常物品管理

详见 [InfiniteSyringes/README.md](InfiniteSyringes/README.md)

### ScopeSensitivity - 更好的倍镜

**版本**: 2.0.0  
**作者**: 不明嚼栗  
**描述**: 倍镜优化 Mod，提供灵敏度、后座力和操作方式三重优化

**功能**:
1. **降低高倍镜开镜灵敏度**：降低 4倍镜、8倍镜开镜时的鼠标灵敏度，使其与 2倍镜接近
2. **优化后座力分布**：根据瞄准距离调整后座力，使近距离和远距离的后座相对视野比例更一致
3. **重做倍镜操作方式**：取消开镜自动漂移，改为鼠标跟随+边缘滚动的控制方式
   - 鼠标跟随：移动鼠标实时调整镜头偏移
   - 边缘滚动：鼠标靠近屏幕边缘时自动扩展视野
   - 回拉跟随：准星往回移动时镜头加速跟随（水平5倍，竖直8倍）
   - 范围限制：限制镜头移动范围与原游戏不同倍镜视野范围一致

详见 [ScopeSensitivity/README.md](ScopeSensitivity/README.md)
